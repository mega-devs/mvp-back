# Mailer System

Система массовой рассылки email с поддержкой SMTP, прокси и шаблонов.

## Возможности

- Управление SMTP серверами
- Поддержка SOCKS5 прокси
- Шаблоны писем с HTML
- Массовая рассылка
- WebSocket уведомления
- Асинхронные задачи
- API интерфейс
- Экспорт/импорт данных

## Технологии

- Python 3.11
- Django 4.2
- Django REST Framework
- Celery
- Redis
- MySQL
- WebSocket (Channels)
- Docker
